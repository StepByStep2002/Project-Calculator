# Project-Calculator

This is a final project of The Foundations course in The Odin Project curriculum.

This calculator is capable of performing basic mathematic operations, such as adding, substracting, multiplying and dividing.

Additionally, it supports float numbers with presicion up to 3 numbers after decimal point, has functionality 
to delete inputs and you can do all of the mentioned operations through keyboard input.

Entering operator again, when both numbers were inputed or when the result is shown, will result in using the 
result of the current operation as the first number for the next operation.

Entering any number, after the result was shown through '=' button or key, will clear both numbers and display to
begin an input for the new operation.


Instruction to keyboard support:

0 - 9: Entering numbers.
'Shift' + '=': Enter 'add' operator.
'-': Enter 'substract' operator.
'Shift' + '8': Enter 'multiply' operator.
'/': Enter 'divide' operator.
'=': Show the result of an operation.
'.': Add decimals to the number.
'Backspace': Delete input.
